## 🐛 **问题根源分析**

你完全正确！问题确实出在 `update-content.mjs` 脚本上。

### **问题原因**：
当你在管理后台编辑内容并导出配置后，运行 `deploy-update.bat` 会调用 `update-content.mjs` 脚本。这个脚本在将 JSON 转换为 TypeScript 代码时，**错误地将字符串数组拆分成了字符对象**。

### **错误的数据转换**：
```typescript
// 输入（正确的 JSON）
tags: ['效率工具', 'React', 'Node.js']

// 错误的输出（之前的脚本）
tags: [
  { '0': '效', '1': '率', '2': '工', '3': '具' },
  { '0': 'R', '1': 'e', '2': 'a', '3': 'c', '4': 't' },
  ...
]
```

这导致 React 无法正确渲染页面，显示空白！
